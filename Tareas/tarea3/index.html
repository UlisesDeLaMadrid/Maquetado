<!DOCTYPE>

Indica que el documento esta bajo el estandar HTML5

<html>

Representa la raiz de un documento HTML, es una buena practica idicar el idioma de la pagina mediante el atributo lang=""

<!--comentario--!>

Define un comentario dentro del codigo, el cual solo sera visible para el desarrollador.

<head>

Incluye la coleccion de meta datos sobre el documento y los enlaces a scripts y hojas de estilo.

<title>

Define el titulo del documento.

<link>

Se ultiliza para enlazar documentos externos de JavaScript y CSS.

<meta>

Definen los metadatos acerca del documento HTML, estas siempre se encuentran dentro de la etiqueta head.

<style>

Permite escribir estilos CSS dentro del documento HTML.

<header>

Indica el contenido dentro de la cabecera de la pagina.

<nav>

Indica que es la seccion donde se encuentra la barra de navegacio.

<main>

Almacena el contenido principal del documento, solo debe de haber uno.

<section>

Define las secciones dentro de la pagina.

<article>

Define secciondes dentro de la etiqueta <section>

<aside>

Define y almacena la barra lateral.

<footer>

Define el pie de pagina y almacena la parte final del documento.

<a>

Esta etiqueta agrega un enlace, su atributo href="" indica el enlace a donde nos dirigira, su sintaxis es la siguiente: <a href="URL">texto del enlace</a>

<p>

Esta etiqueta sirve para agregar parrafos de texto, su sintaxis es la siguiente: <p>Texto del parrafo</p>

<h>

Esta etiqueta nos sirve para agregar titulos de diferentes tamaños, y va desde el <h1> hasta el <h6>, siendo el <h1> el titulo de mayor tamaño y el <h6> el de menor tamaño, su sintaxis es la siguiente: <h1>Titulo<h1>

<img>

Esta etiqueta sirve para agregar una imagen dentro de nuestra pagina, es una etiqueta de cierre automatico por lo cual no necesita cerrarse con "/", su atributo src="" nos indica la ruta de la imagen y su atributo alt="" es una descripcion corta de la imagen que se muetra en caso que la imagen no se cargue correctamente, su sintaxis es la siguiente: <img src="Ruta de la imagen" alt="Descripcion de la imagen">

<span>

Es una etiqueta de contenedor en linea, nos sirve para dar estilo a un texto.
